# 🚀 BrainCoach AI - Quick Start Guide

## ⚡ Get Started in 3 Minutes

### Step 1: Start the Server ✅
The server is already running at: **http://localhost:5001**

### Step 2: Create Your Account
1. Open your browser and go to `http://localhost:5001`
2. Click **"Get Started"** or **"Register"**
3. Fill in your details:
   - Full Name
   - Email
   - Phone (optional)
   - Password
4. Click **"Start Free Journey ✨"**

### Step 3: Add Your First Child
1. After logging in, you'll see the Dashboard
2. Click **"+ Add a Child"** button
3. Enter:
   - Nickname (e.g., "Little Ninja")
   - Age (3-18 years)
   - Grade (optional)
4. Click **"Create Profile ✨"**

### Step 4: Start Training!
1. Find your child's card on the dashboard
2. Click the **⋮** menu button
3. Select **"🎯 Start Training"**
4. **Allow webcam access** when prompted
5. Click **"Start the Adventure! 🚀"**
6. Follow the on-screen instructions!

## 🎮 How to Use the Training Session

### During Training:
- **Watch the left screen** - See the demonstration
- **Follow along** - Copy the movements with your hand
- **Check the accuracy ring** - Green = Good! Red = Try again!
- **Listen for sounds** - 🌟 Success sound when doing great!
- **Watch the timer** - Session duration countdown

### Controls:
- **Start Button** - Begin the training session
- **Stop Button** - End session early (if needed)
- **Resize Divider** - Drag the middle bar to adjust screen sizes

## 📊 Understanding the Dashboard

### Focus Score Card
- Shows average accuracy across all sessions
- Updates in real-time after each session
- Green arrow indicates improvement

### Sessions Completed Card
- Total number of training sessions
- Tracks consistency

### Improvement Journey Chart
- Visual graph of last 7 sessions
- Shows progress over time
- Helps identify trends

### Trophy Case 🏆
- **🌱 First Steps** - Complete your first session
- **👑 Concentration King** - Achieve 90%+ accuracy
- **🔥 Consistency Master** - Train 7 days in a row
- **💯 Perfect Score** - Get 100% accuracy

### Level System
- Level 1: 0-4 sessions
- Level 2: 5-9 sessions
- Level 3: 10-14 sessions
- And so on... (Level up every 5 sessions!)

## 🎯 Tips for Best Results

### For Parents:
1. **Consistent Schedule** - Train at the same time daily
2. **Short Sessions** - Start with 1-2 minutes
3. **Positive Reinforcement** - Celebrate small wins!
4. **Track Progress** - Review the dashboard weekly
5. **Download Reports** - Keep PDF records

### For Children:
1. **Good Lighting** - Ensure room is well-lit
2. **Clear Background** - Stand in front of plain wall
3. **Camera Position** - Face the camera directly
4. **Hand Visibility** - Keep hand in frame
5. **Have Fun!** - It's a game, enjoy it!

## 🔧 Troubleshooting

### "Camera Not Working"
✅ **Solution**: 
- Click the camera icon in browser address bar
- Select "Allow" for camera access
- Refresh the page

### "Hand Not Detected"
✅ **Solution**:
- Ensure good lighting
- Move hand closer to camera
- Keep hand open and visible
- Try different background

### "Low Accuracy Score"
✅ **Solution**:
- Watch the demo video carefully
- Move slowly and deliberately
- Keep hand steady
- Practice makes perfect!

### "Page Won't Load"
✅ **Solution**:
- Check if server is running
- Clear browser cache
- Try different browser
- Check internet connection
